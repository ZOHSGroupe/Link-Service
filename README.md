## Description

Link-Service is an Application Programming interface for add,update,delete and get links of ZOHS company.
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Dockerize the app :

```bash
# docker-compose
$ docker compose up
```
- Open pgAdmin in your browser : [PgAdmin](http://localhost:5050)
